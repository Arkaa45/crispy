### Cara Menggunakan File Ini di Microsoft Word:
1. Buka file dokumen kosong di Microsoft Word.
2. Anda dapat memblok seluruh tabel di atas dari awal header `| No |` hingga baris terakhir.
3. Lakukan **Copy** (Ctrl+C), lalu **Paste** (Ctrl+V) langsung ke dokumen Microsoft Word. Word akan secara otomatis mendeteksi format tabel ini dan mengaturnya dengan rapi.
4. (Opsional) Di Microsoft Word, klik tabel tersebut, pilih **Table Design / Layout**, dan klik **AutoFit > AutoFit to Window** agar lebar kolom menyesuaikan dengan halaman Anda secara optimal.
